FAQ Interaktif

Apa itu jQuery? ▶
JQuery juga merupakan library JavaScript yang dapat mempercepat proses pembuatan website dengan sintaks yang tidak terlalu rumit, kodenya ringkas, dan fiturnya lengkap. jQuery membuat pemrosesan di HTML, seperti event handling, animasi, perubahan dan manipulasi dokumen, serta Ajax dapat menjadi lebih sederhana.
Bagaimana cara menggunakan .click() di jQuery? ▶
Metode .click() digunakan untuk melampirkan penangan acara "click" ke elemen HTML yang dipilih. Sintaks dasarnya adalah $("selector").click(function() { ... });.
Apa fungsi dari .toggle()? ▶
Metode .toggle() digunakan untuk menampilkan atau menyembunyikan elemen yang cocok. Jika elemen ditampilkan, elemen tersebut disembunyikan; jika disembunyikan, elemen tersebut ditampilkan. Ini sangat berguna untuk menu dropdown atau, seperti dalam kasus ini, jawaban FAQ.
Apa perbedaan antara .hover() dan CSS :hover? ▶
CSS :hover murni untuk tujuan gaya visual sederhana. .hover() jQuery adalah fungsi pintasan untuk mengikat dua penangan acara: satu untuk mouseenter (saat masuk) dan satu lagi untuk mouseleave (saat keluar), memungkinkan Anda menjalankan logika JavaScript yang lebih kompleks.
Apakah jQuery masih relevan di tahun ini? ▶
Meskipun kerangka kerja JavaScript modern seperti React, Vue, dan Angular lebih populer untuk aplikasi kompleks, jQuery masih digunakan secara luas di jutaan situs web dan sangat relevan untuk proyek yang membutuhkan manipulasi DOM yang cepat dan sederhana.
